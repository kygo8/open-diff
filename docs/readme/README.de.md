# Open Diff

Open Diff ist ein plattformuebergreifendes Werkzeug zum Vergleichen und Zusammenfuehren von Dateien, Ordnern, Tabellen, Bildern und Binaerdaten.

## Funktionen

- Textvergleich: Unterschiede pro Zeile, Unterschiede pro Zeichen, Syntaxhervorhebung und Ignorierregeln.
- Ordnervergleich: rekursives Scannen, Differenzstatus, Filter und grundlegende Dateioperationen.
- Ordnersynchronisierung: Vorschau von Kopier- und Loeschaktionen vor der Ausfuehrung.
- Tabellenvergleich: CSV, TSV und tabellenartige Daten.
- Bildvergleich: Pixelvergleich, Differenzanteil und optionaler Alpha-Kanal.
- Binaervergleich: Hex-Ansicht, Offsets und Hervorhebung unterschiedlicher Bytes.
- Drei-Wege-Merge: Konflikterkennung und Konfliktloesung mit base, left und right.
- Sitzungsverwaltung: Sitzungen speichern, erneut oeffnen, loeschen, importieren und als JSON exportieren.
- Berichte: HTML-Berichte generieren.
- Automatisierung: Workflows per Kommandozeile und Skript.

## Anwendungsfaelle

- Quellcodeaenderungen pruefen.
- Konfigurationsdateien vergleichen.
- Projektordner synchronisieren.
- CSV- oder Tabellendaten vergleichen.
- Unterschiede in Bildern und Binaerdateien untersuchen.
