# Open Diff

Open Diff es una herramienta multiplataforma para comparar y fusionar archivos, carpetas, tablas, imagenes y datos binarios.

## Funciones

- Comparacion de texto: diferencias por linea, diferencias por caracter, resaltado de sintaxis y reglas de ignorado.
- Comparacion de carpetas: escaneo recursivo, estados de diferencia, filtros y operaciones basicas de archivos.
- Sincronizacion de carpetas: vista previa de acciones antes de copiar o eliminar.
- Comparacion de tablas: CSV, TSV y datos con estructura de hoja de calculo.
- Comparacion de imagenes: comparacion por pixel, porcentaje de diferencia y canal alpha opcional.
- Comparacion binaria: vista hexadecimal, offsets y resaltado de bytes diferentes.
- Fusion de tres vias: deteccion y resolucion de conflictos con base, left y right.
- Gestion de sesiones: guardar, reabrir, eliminar, importar y exportar sesiones en JSON.
- Informes: generacion de informes HTML.
- Automatizacion: flujos de trabajo por linea de comandos y scripts.

## Casos de uso

- Revisar cambios de codigo fuente.
- Comparar archivos de configuracion.
- Sincronizar carpetas de proyecto.
- Comparar datos CSV o tabulares.
- Inspeccionar diferencias en imagenes y archivos binarios.
