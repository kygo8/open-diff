# Open Diff

Open Diff는 파일, 폴더, 표 데이터, 이미지, 바이너리 데이터를 비교하고 병합하기 위한 크로스 플랫폼 도구입니다.

## 기능

- 텍스트 비교: 줄 단위 차이, 문자 단위 차이, 문법 강조, 무시 규칙.
- 폴더 비교: 재귀 스캔, 차이 상태 표시, 필터, 기본 파일 작업.
- 폴더 동기화: 실행 전 작업 미리보기, 단방향 업데이트, 양방향 업데이트, 미러 동기화.
- 표 비교: CSV, TSV 및 스프레드시트 형태 데이터 비교.
- 이미지 비교: 픽셀 단위 비교, 차이 비율, Alpha 채널 비교.
- 바이너리 비교: 16진수 보기, 오프셋 표시, 차이 바이트 강조.
- 3-way merge: base, left, right 버전을 사용한 충돌 감지와 해결 지원.
- 세션 관리: 비교 세션 저장, 다시 열기, 삭제, JSON 가져오기/내보내기.
- 보고서: HTML 비교 보고서 생성.
- 자동화: 명령줄과 스크립트 기반 반복 작업 지원 예정.

## 사용 예

- 소스 코드 변경 검토.
- 설정 파일 비교.
- 프로젝트 폴더 동기화.
- CSV 또는 표 데이터 비교.
- 이미지와 바이너리 파일 차이 확인.

## 상태

Open Diff는 초기 단계의 프로젝트입니다. 설치, 빌드, 기여 가이드는 이후 추가될 예정입니다.
