# Open Diff

Open Diff 是一個跨平台差異比較與合併工具，用於比較檔案、資料夾、表格、圖片和二進位資料。

## 功能說明

### 文字比較

- 逐行比較兩個文字檔案。
- 標示新增、刪除、修改內容。
- 支援修改行內的字元級差異。
- 支援常見程式語言和標記語言的語法醒目提示。
- 支援忽略空白、大小寫和註解差異。
- 支援差異導覽、搜尋、跳至行號、顯示行號和自動換行。
- 支援選擇檔案編碼。

### 資料夾比較

- 遞迴掃描左右資料夾。
- 顯示相同、已修改、僅左側存在、僅右側存在的檔案。
- 支援依大小、修改時間或內容校驗值進行比較。
- 支援依狀態篩選顯示結果。
- 支援排除指定檔案模式。
- 支援從資料夾比較中開啟檔案級比較。
- 支援基礎複製操作。

### 資料夾同步

- 執行前預覽複製和刪除動作。
- 支援向右更新、向左更新、雙向更新。
- 支援向右鏡像和向左鏡像。
- 顯示同步進度和操作記錄。

### 表格比較

- 支援 CSV、TSV 和類電子試算表資料。
- 偵測新增、刪除、修改和相同行。
- 標示修改行中的儲存格差異。
- 支援表格檢視和統一檢視。
- 支援設定分隔符號和表頭。

### 圖片比較

- 左右並排載入兩張圖片。
- 執行像素級比較。
- 顯示差異像素比例。
- 支援檢查圖片尺寸不一致。
- 可選擇是否比較 Alpha 通道。

### 二進位與十六進位比較

- 載入兩個二進位檔案。
- 顯示偏移位址、十六進位位元組和 ASCII 內容。
- 標示差異位元組。
- 標記僅左側或僅右側存在的位元組範圍。

### 三方合併

- 支援 base、left、right 三個版本。
- 偵測衝突修改。
- 顯示已解決和衝突的輸出區段。

### 工作階段管理

- 儲存比較工作階段。
- 查看最近工作階段。
- 重新開啟歷史工作階段。
- 刪除舊工作階段。
- 匯入和匯出 JSON 工作階段資料。

### 報告與自動化

- 產生 HTML 比較報告。
- 支援適合列印的報告樣式。
- 規劃支援命令列和腳本化比較任務。

## 典型使用情境

- 程式碼變更審查。
- 設定檔比較。
- 專案資料夾同步。
- 發佈套件差異檢查。
- CSV 或表格資料比較。
- 圖片和二進位檔案差異檢查。
